# DMF 48通道数字微流控控制器上位机

## 项目简介
DMF 48通道数字微流控控制器上位机是一套面向微流控液滴操控场景的可视化控制软件。项目基于 Python + PyQt5 构建，提供串口通信、48电极网格显示、BFS 自动寻路以及液滴运动控制能力，适用于实验室与工业化微流控平台的上位机接入与调试。

本项目的核心目标是：
- 实现 DMF 液滴操控过程的图形化监控与控制
- 通过串口协议与底层控制板进行实时通信
- 利用 BFS 路径规划算法自动驱动电极完成液滴路径导航

## 项目功能清单
### 1. 串口通讯
- 扫描并连接本机串口设备
- 支持串口状态监控与异常提示
- 可向底层控制板下发电极控制命令

### 2. 可视化电极网格
- 6 行 × 8 列，共 48 个电极单元的网格显示
- 支持电极状态高亮、起点/终点/障碍物标记
- 便于直观观察液滴路径与电极布局

### 3. BFS 自动寻路
- 基于广度优先搜索（BFS）算法实现最短路径规划
- 可避开障碍电极，生成稳定可执行的路径序列

### 4. 一键液滴移动
- 支持路径执行与分步驱动
- 可设置延迟参数进行液滴移动控制与实验验证

### 5. 状态图例说明
- 空闲电极：可用状态
- 起点电极：液滴起始位置
- 终点电极：液滴目标位置
- 障碍电极：不可通行区域
- 运行路径：当前规划轨迹显示

## 运行环境
- Python 3.9 及以上
- PyQt5
- pyserial
- qdarkstyle（可选，界面主题增强）

## 部署与启动
### 1. 安装依赖
在项目根目录执行：

```bash
pip install -r requirements.txt
```

### 2. 启动应用
运行主程序：

```bash
python main.py
```

## 说明与适用场景
该项目适用于：
- 微流控液滴操控实验平台
- 48 通道电极驱动控制系统
- 上位机调试、路径验证与液滴自动移动演示

如需进一步扩展，可继续增加路径参数配置、液滴轨迹回放、日志导出与设备协议适配功能。
