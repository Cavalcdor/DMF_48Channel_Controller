# DMF 48通道数字微流控上位机
## 项目说明
软件：PyQt5上位机，6行×8列=48电极，对接Arduino Mega2560+3片MCP23017(0x27/0x25/0x24)
开发：翁超然 | 硬件：陈若涵、林芯羽 | 实验：周钰
## 1.通信参数
串口波特率：115200，8N1
下发指令：ON,n / OFF,n / ALLON / ALLOFF / TEST / LIST
n范围：0~47
## 2.坐标映射规则
网格(r行,c列) → 硬件编号 n = r*8 + c
r∈[0,5]，c∈[0,7]，总计48点位
## 3.功能清单
1.网格单点开关控制电极
2.标记起点/终点/障碍物(故障电极)
3.BFS自动路径规划
4.多液滴路径避让，无电极冲突
5.自动分步下发指令驱动液滴移动
