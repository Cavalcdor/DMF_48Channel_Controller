## 📖 项目简介

基于 Python + PyQt5 构建的 **DMF（数字微流控）48通道控制器上位机**，提供电极网格可视化、A\* 智能寻路、多液滴并行路径规划、芯片测试（坏点自动绕路扫描）与串口实时控制等核心功能，面向微流控液滴操控实验场景。

## ✨ 功能特性

### 🖥️ 学术风 UI
- **三栏布局**：左侧 QTabWidget（控制面板 + 芯片测试）、中央网格视图、右侧状态面板
- **学术风扁平设计**：无圆角、无卡片阴影、灰/蓝色配色系统
- **实时状态指示**：顶栏显示串口连接状态与运行状态

### 🔌 串口通讯
- 自动扫描本机串口，连接状态实时监控
- 支持协议：`ON,ch` / `OFF,ch` / `ALLON` / `ALLOFF` / `TEST` / `LIST` / `HELP`
- 快捷指令 + 自定义指令输入 + 单步通道测试
- **虚拟串口模式**：无需硬件即可模拟全部指令响应，方便调试与演示

### 🧩 可视化电极网格
- 自定义行列数（2×2 至 16×16），默认 6×8（48 电极）
- 六种状态：空闲 / 起点(蓝) / 目标(橙) / 障碍物(深灰) / 执行高亮(绿框+▶N) / 芯片测试覆盖
- 交互模式：起点/终点模式 ↔ 障碍物模式
- 智能点击：空闲格自动设为起点或目标，已设格点击取消

### 🧠 路径规划
- **A\* 算法**（曼哈顿距离启发式）四方向寻路
- **多液滴并行规划**（最多 8 个液滴），贪心排序避免路径冲突，支持液滴融合（共享目标点）
- 失败诊断：区分障碍物阻挡、其他液滴占用、起点等于终点等场景
- **规划路径**：仅计算预览，不下发串口
- **运行路径**：计算后自动串口步进执行
- **单步执行**：手动逐步前进，适合调试观察

### 💧 液滴管理
- 支持 1～8 个液滴，独立编号与路径颜色标识
- 液滴融合：多个液滴汇聚到同一个终点
- 导航切换：◀上一个 / 下一个▶ / ⏹回到液滴1
- 清除操作：✕清除当前（单个）/ ✕清除所有（全部）

### ⚡ 路径执行
- 可配置步长延迟（100 / 200 / 500 / 1000 ms）
- 定时器驱动逐级开关电极（ON 当前步、OFF 上一步）
- 网格实时高亮：绿色边框 + ▶N 标注当前液滴位置
- 停止按钮中断全部继电器
- 支持自动执行 / 单步手动执行

### 🔬 芯片测试（坏点检测）
- 独立标签页，专用于检测芯片上 48 个电极的好坏
- **蛇形扫描路径**：偶数行左→右，奇数行右→左，覆盖全部电极
- **坏点自动绕路**：A\* 寻路绕过已标记坏点，无需手动引导
- 自动 / 单步两种测试模式，可调间隔 50~5000ms
- 实时统计：通过/坏点/未测计数 + 绕路次数
- 可视化覆盖：通过（绿✓）、坏点（红✗）、当前（蓝框）
- 结果导出支持 TXT / CSV 格式

### 📊 信息面板
- 右侧图例 + 路径规划结果详情
- 液滴配对状态总览 + 路径信息

### ↩️ 撤销/重做
- 完整网格操作历史（最多 100 步）
- Ctrl+Z / Ctrl+Y，覆盖起点/终点/障碍物/途经点操作

### 📸 网格截图导出
- 一键导出网格视图为 PNG 图片
- 支持复制到剪贴板

### 🔍 日志搜索过滤
- 关键词实时搜索 + 级别复选框过滤（DEBUG/INFO/SUCCESS/WARN/ERROR）
- 动态计数显示

### 📹 操作录制/回放
- 录制串口指令序列，支持回放和导出 JSON

### 🔔 完成通知音效
- 路径执行和芯片测试完成时播放提示音
- 可通过编辑菜单开关

### 🧩 途经点（路径微调）
- 右键设置途经点（菱形◇标记），路径依次经过
- 支持同一液滴多途经点

## 🎮 使用流程

### 标准液滴操控流程

```
连接串口 → 配置网格 → 设置起点/终点 → 设置障碍物 → 规划路径 → 运行路径
```

1. **连接串口** — 下拉选择端口，点击「连接」
2. **配置网格** — 调整行/列数，点击「新建网格」
3. **设置液滴** — 切换液滴编号，在网格上点击设置起点（蓝）和终点（橙）
4. **设置障碍物** — 切换到障碍物模式，标记不可通行电极
5. **规划路径** — 点击「规划路径」预览路径
6. **运行路径** — 确认无误后点击「运行路径」自动执行

### 芯片测试流程

```
连接串口 → 切换到「芯片测试」标签 → 自动测试 → 标记坏点 → 自动绕路 → 导出结果
```

1. **连接串口** — 确保硬件已连接
2. **切标签** — 左侧面板切换到「芯片测试」标签页
3. **自动测试** — 点击「自动测试」启动蛇形扫描
4. **标记坏点** — 液滴到达异常电极时点击「坏点」
5. **自动绕路** — 系统自动 A\* 绕路跳过坏点
6. **导出结果** — 测试完成后导出 TXT / CSV 报告

### 补充说明

- 新建/重置网格后自动回到液滴 1，无需手动切换
- 调整网格大小时行列参数会同步到全局配置，确保寻路和索引映射正确
- 「清除障碍物」只清黑色格子；「清除当前」只清当前液滴的蓝/橙格；「清除所有」清全部液滴配置
- 芯片测试的坏点标记不影响正常液滴操作的障碍物设置
