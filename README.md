## 📖 项目简介

DMF 48通道数字微流控控制器上位机是一套面向微流控液滴操控场景的**商业级可视化控制软件**。基于 Python + PyQt5 构建，采用现代化 UI 设计语言，提供串口通信、48 电极网格可视化、A* 智能寻路、多液滴并行路径规划等核心功能。

本项目的核心目标：
- 🎯 实现 DMF 液滴操控全流程图形化监控与控制
- 🔌 通过串口协议与底层控制板实时通信
- 🧠 利用 A* 路径规划算法驱动电极完成液滴路径导航
- 🎨 提供清爽、直觉化的用户交互体验

## ✨ 功能特性

### 🖥️ 现代化 UI
- **深色顶栏 + 三栏布局**：左侧操作面板、中央网格视图、右侧状态面板
- **商业级视觉风格**：统一配色系统、圆角控件、柔和平滑的交互反馈
- **实时状态指示**：顶栏显示连接状态与运行状态

### 🔌 串口通讯
- 自动扫描并连接本机串口设备
- 串口状态实时监控与异常提示
- 支持 ALLON / ALLOFF / 单通道开关 / TEST / LIST / HELP 协议
- 一键快捷测试（ALLON、ALLOFF、TEST、LIST、HELP）

### 🧩 可视化电极网格
- 自定义行列数（2×2 至 16×16），默认 6×8 共 48 电极
- 四种电极状态：空闲 / 起点 / 目标 / 障碍物
- 交互模式切换：起点/终点模式 ↔ 障碍物模式
- 智能点击逻辑：重复点击自动轮换状态
- 电极圆角渲染，支持动态字体缩放

### 🧠 路径规划
- **A\* 算法**（曼哈顿距离启发式）高效寻路
- 支持**多液滴并行路径规划**（最多 8 个液滴）
- 路径冲突检测（交叉/重叠/追尾）
- 详细的失败诊断信息（5 种失败原因上报）
- **规划路径**模式：仅计算不执行，安全预览
- **运行路径**模式：计算后自动串口下发执行

### 💧 多液滴管理
- 支持 1~8 个液滴独立设置起点/终点
- **「下一个液滴」**快速切换
- 独立编号与颜色标识
- 上一滴/下一滴顺序切换

### ⚡ 路径执行控制
- 可配置步长延迟（100ms / 200ms / 500ms / 1000ms）
- 实时停止中断（断开所有继电器）
- 执行状态在顶栏与右侧面板同步显示

### 📊 信息面板
- 右侧状态面板：图例说明 + 实时路径信息
- 液滴配对状态一览

## 🛠️ 技术架构

```
DMF_48Channel_Controller/
├── main.py                  # 主程序入口 + UI 编排 + 业务逻辑
├── requirements.txt         # Python 依赖清单
├── README.md                # 项目说明文档
├── assets/                  # 样式文件与界面资源
│   └── style.qss
└── src/                     # 核心模块目录
    ├── global_cfg.py        # 全局配置与参数定义
    ├── grid_widget.py       # 电极网格可视化组件（QWidget + paintEvent）
    ├── path_algorithm.py    # A* 路径规划算法
    └── serial_driver.py     # 串口通信驱动（QThread）
```

### 核心模块说明

| 模块 | 文件 | 职责 |
|------|------|------|
| **主窗口** | `main.py` | 布局管理、信号/槽连接、路径执行、串口控制 |
| **网格组件** | `src/grid_widget.py` | 电极渲染(paintEvent)、交互点击、自动缩放、液滴配对 |
| **路径算法** | `src/path_algorithm.py` | A* 寻路、多液滴规划、冲突检测、失败诊断 |
| **串口驱动** | `src/serial_driver.py` | 串口扫描、异步通信、ALLON/ALLOFF 命令 |
| **全局配置** | `src/global_cfg.py` | 默认电极数、通信参数等 |

## 🎮 使用指南

### 基本流程

1. **连接串口** → 选择串口号，点击连接
2. **配置网格** → 设置行列数，点击新建网格
3. **设置起点/终点** → 切换到「起点/终点」模式，点击网格设置
4. **设置障碍物** → 切换到「障碍物」模式，标记不可通行电极
5. **规划路径** → 点击「规划路径」预览计算结果
6. **运行路径** → 确认无误后点击「运行路径」执行

### 交互模式

- **起点/终点模式**：单击空闲格设为起点，再次单击同格轮换为目标，继续单击回到空闲
- **障碍物模式**：单击切换障碍物状态
- 两种模式互斥，点击切换按钮自动切换

### 多液滴操作

- 使用 ▲/▼ 或「下一个液滴」切换当前编辑的液滴编号
- 每个液滴可独立设置起点和目标位置
- 路径规划时自动为所有已配对的液滴规划路径
